# Getting Started

## How to run

### Locally

* Build the project:
```
./mvnw clean package
```
Build skipping tests
```
./mvnw clean package -DskipTests
```
* Execute the project
```
java -jar target/5e.exptracker-0.0.1-SNAPSHOT.jar
```

The API will be available at [localhost:8080](http://localhost:8080).
